User Guide

Welcome to the Loan Management System! This guide will help you navigate and use the system effectively.

Dashboard

The dashboard provides an overview of your loans, assets, and accounts. Use the navigation bar to access different sections.

Loans

  • View Loans: See a list of your current loans.
  • Apply for Loan: Use the loan application form to request a new loan.
  • Loan Status: Track the status of your loan applications.

Assets

  • Manage Assets: Add, update, or delete assets linked to your loans.
  • Upload Documents: Attach relevant documents to your assets.

Accounts

  • Manage Borrowers: View and edit borrower information.
  • Create New Borrowers: Add new clients to the system.

Support

If you need assistance, please contact your system administrator or support team.
